Real‑Time PID Control of a Motor‑Driven Cart Using MATLAB/Simulink

OCT 2025 – DEC 2025

This project presents the design and implementation of a real-time control system for a motor‑driven cart moving on a belt. The cart is physically constructed, equipped with a DC motor and an encoder, and controlled directly through MATLAB/Simulink using live input–output signals. The work includes developing a control‑oriented mathematical model of the cart’s electromechanical dynamics, estimating key parameters from experimental motion data, and validating the model against real measurements to ensure reliable controller design. A PID controller is then implemented to command the cart’s movement and bring it to precise stops at target positions, achieving fast response, minimal overshoot, and consistent performance in both simulation and hardware tests.
